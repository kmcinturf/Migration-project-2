The purpose of this project was to study and analyze the migration paths of different animal species e.g. Blue Fin Whales, Blue Whales, and the Argentine Barn Swallows. To acomplish this analysis, we first bluit three different sets of dataframes with the data pulled from the following data: Blue Fin Whales, Argentine Barn Swallows, and Blue Whales

After assembling the dataset, we used Leaflet to plot multiple sets of data from each animal species based on their locatoin in order to track their migration paths through out the year. Afterwards we created an animation for each species showing the actual movement between where the animal started the year to where it ended.
